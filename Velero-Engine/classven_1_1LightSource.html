Detailed Description

Object that emits light within a lighting layer.

These objects are meant to be used within a LightingLayer. They can also be used within a regular layer, but the lighting effect would have some limitations.

void ven::LightSource::setGlow ( bool  glow)

Set the value of the glow flag.

When the glow is inactive, the LightingLayer will only use the light to reveal the area under the fog. If it is active, it will also add the color to the image. Note that when the glow is not set, the light won't be visible if the fog opacity is 0.

Left: glow deactivated with intensity at 1 and 0.5. Right: glow activated with intensity at 1 and 0.5.

glowValue to set the flag.
void ven::LightSource::castLight ( )

Calculates the area that should be iluminated with a ray casting algorithm.

For the calculations, the segments from m_ptrSegmentPool are used. If the fog opacity is not 0, then LightingLayer::updateFog should be called somewhere between this function and the next draw.
