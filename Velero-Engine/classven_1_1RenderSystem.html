Detailed Description

System to group layers, manage and render them.

This system consists on a stack of layers and a view, that can be thought of as the camera. By default the layers are dynamic, but can be made static by locking them. the operations that return a layer have an equivalent returning a lighting layer, making a static cast without type checking. Using a dynamic layer as a lighting layer is not that dangerous, but the other way around can produce an undefined behaviour, so the user must differentiate them well.

For such purpose, all layers must have a name, and it is recommendable to always access and modify them using it, to make clear the nature of each layer.

Also, each layer of the system has its own render states, so a transform, shader, blend mode or even texture can be applied to it.
