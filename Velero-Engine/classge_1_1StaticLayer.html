Detailed Description

Layer that doesn't allow changes.

This layer buffers the renderables added to it, and requires a manual update to show them. It is thought to be updated once and not to be changed after that. If you wanted to make a second update and keep the initial renderables, they would have to be added again.

This layer is meant for grouping objects that won't be updating and keep the draw time constant, no matter the number of objects.
