◆ clear()

void candle::Lighting::clear ( )

Removes all lights.

This function doesn't delete the pointers, only forgets them. Also, the edge pool is not deleted. To clear it, you must explicitly clear m_edgePool.

◆ updateFog()

void candle::Lighting::updateFog ( )

Applies fog changes and updates the field of view of the lights.

If the fog is visible and the lights move, this function is required to keep the fog coherent with the lights.
