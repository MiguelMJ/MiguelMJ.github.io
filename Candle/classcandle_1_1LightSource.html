Detailed Description

Object that emits light within a Lighting object.

These objects are meant to be drawn as part of a Lighting object.

Member Function Documentation

void candle::LightSource::setIntensity ( float  intensity)

Set the light intensity.

The intensity of the light determines two things: how much fog opacity it reduces, and how much of its color is added to the layers below when the glow is active.

Parameters
intensityValue from 0 to 1. At 0 the light is invisible.

void candle::LightSource::setGlow ( bool  glow)

Set the value of the glow flag.

When the glow is inactive, the Lighting will only use the light to reveal the area under the fog. If it is active, it will also add the color to the image. Note that when the glow is not set, the light won't be visible if the fog opacity is 0.

Parameters
glowValue to set the flag.

void candle::LightSource::castLight ( )

Calculates the area that should be iluminated with a ray casting algorithm.

For the calculations, the segments from m_ptrSegmentPool are used. If the fog opacity is not 0, then Lighting::updateFog should be called somewhere between this function and the next draw.
